## 🚀 快速安装

如果你想在你的 Arch Linux 上复刻我的配置，只需按照以下步骤操作：

### 1. 克隆仓库

```bash
git clone git@github.com:CAO-PNG/Arch-hyprland.git
cd Arch-hyprland

2. 运行安装脚本

该脚本会自动安装 yay、必要的依赖包，并建立配置文件软链接。
Bash

chmod +x install.sh
./install.sh
