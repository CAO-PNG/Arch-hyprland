这里是关于大六壬框架的搭建问题，主要是清楚的认识`天地盘`，`四课`，`三传`，建立一个比较清晰的概念，至于起课，立四课，出三传，随便都可以搜到

# 天地盘

如图

```bash
巳午未申
辰	 酉
卯	 戌
寅丑子亥
```

像这样的结构就是地盘，地盘是不动的，而天盘是怎么出现的呢？ 

首先我们需要知道一点天文的小知识：太阳所在的平面我们称之为黄道平面，而地球所在的平面我们称之为赤道平面，两个平面一定会有一个交点，这个交点会锚定一段时空，因此大六壬的四课，三传就会衍生出来，进而推断人事。

古人是以赤道平面作为坐标轴的，因为这样地球的轨迹近似于圆形，相对准确些，所以，对于地球人来说，地盘就是不变的，而大六壬的起课方式：月将加时，就是为了找出赤道和黄道交点，(简而言之，地盘是赤道平面，天盘是黄道平面，月将加时的目的是找到两者的交点，这样就可以用锚定的这片时间来推测事了)

简单说一下天地盘的样子，月将加时：现在的时间`2026-1-11-14-04`，转换成干支：乙巳，己丑，乙酉，甲申 ----> 月将：丑，所以是丑加在申上，

```bash
戌亥子丑
酉	 寅
申	 卯
未午巳辰
```

具体的排法可以看书，或者看视频

这样的就是天盘了，而刚刚说了，天盘是地盘演变出来的，故而地盘是母，天盘是子，而子归于母，，那么就是子是原因，最后化作结果，所以地盘是果，天盘是因，

# 四课

起法就是干支上支，依照子归于母的因果理论，实际上就是不断的去提取日干和日支的原因，分析出正确的，最本质的原因；同理，你可以依照这样的方式去找到对应的结果

# 三传

理气实际上已经在天地盘完成了，但是我们并不知道这其中最重要的那条线是什么，而三传是找到的最重要的那根线。

而三传的起法本质是”克“，而克本质上是阴阳激荡，老阴老阳，少阴少阳之间的互相碰撞，而这四个对应着`金木水火`(这里的顺序不一定是一一对应的)，所以可以看到缺少了一个土，但是五行生克里面还是把土加了进去，这样可能会存在一些问题。

所以，三传可以不用重点看，但是三传一定是矛盾的激发点，因为三传的出现就是`克`，而`克`是事物的矛盾开始的地方

# 一些技法

## 十二长生

关于十二长生：对于世间万物的描述中，光五行是不够的，所以，为了更好更细致的描述出事物的关系，就出现了十二长生，比如：出现`胎`就说明这件事在萌芽中，也可以代表懵懂，出现长生：代表正在茁壮的成长，生命力旺盛，也可以表示学历等等

## 关于空亡

空亡表示的是暂时性的缺失：注意是暂时性的，不是一定失去的，它存在，只是时候未到，机会不成熟，仅仅代表一种象义，而不是代表空，断了，因为空亡的产生是由于干支缺少配对，导致一旬有两天到下一旬，这两天没有消失，只是到了下一旬才会出现

## 天官和地支同位

这个其实就是天官与地支其实是相通的：比如青龙同寅，朱雀同午，可以在一起看，也可以分开看

## 四课的定位

干支定位是为了方便快速进入判断事情的层面，实际上可以以任意字作为太极点，不过，实际上只要我们最开始定好了太极(一般是干支)，后续都是为了这个而分析的，所以一般没什么改的必要，而分析热
